# 🎙️ Voice Effects Changer

## 📌 Overview
Voice Effects Changer is a web-based DSP project that allows users to upload or record audio and apply different voice effects and sound transformations in real time.

The project focuses on Digital Signal Processing (DSP) concepts such as pitch shifting, time stretching, filtering, echo, reverb, and audio mixing.

---

## 🚀 Features

- Upload audio files or record voice directly
- Apply multiple voice effects
- Real-time audio processing
- Support for different audio formats
- Interactive and simple web interface

## 🧠 Project Structure

The backend contains a main function called:

```python
apply_effect()
```

This function handles all presets, characters, and sound effects using:
- `if`
- `elif`

Each effect applies different DSP operations depending on the selected voice mode.
